# Flowbeat 🎵

Plataforma de descubrimiento musical que rompe la burbuja algorítmica.

## ¿Qué hace?
- Elige tu estado de ánimo (energético, melancólico, relajado, eufórico)
- Genera canciones: 70% familiar + 30% géneros sorpresa
- Guarda tus descubrimientos favoritos
- Historial persistente con localStorage

## Tecnologías
- Vite + Vanilla JavaScript
- API de Last.fm (free tier)
- CSS custom sin frameworks

## Cómo correrlo
npm install
npm run dev

Abre http://localhost:5173
